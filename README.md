## Configurando ambiente de desenvolvimento

**Instalar Ides, dependências, etc...**

 - Instalar o git
 - instalar o flutter https://docs.flutter.dev/get-started/install
 - é possível instalar clonando o repositório (git clone https://github.com/flutter/flutter.git -b stable)
 - Criar variáveis de ambiente (adicionar na variável **path** do usuário flutter\bin)
 - instalar Android Studio
 - rodar comando `flutter doctor`

**Caso dê algum erro na instalação do `SDK command-line tools`**
> Just like you, I have Android Studio installed, the key is to install Android SDK Command-line Tools, steps being:
> - Open Android Studio
> - Tools Menu, SDK Manager
> - In the window that comes up there are inner panels, choose SDK Tools panel
> - Tick Android SDK Command-line Tools
> - Choose Apply button near the bottom of the window

**Para iniciar o projeto**

- `flutter create nomeProjeto`
- `cd diretorioDoProjeto` > `fluter run`
- `flutter devices` > ver dispositivos conectados (virtuais e físicos)
- `flutter run` ou `fluter run -d deviceNome`

**com o projeto rodando...**

- `r` > para atualizar a tela do app
- `R` > para reiniciar o app
